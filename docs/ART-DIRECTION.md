# Art Direction Notes

## Visual target
Army Royale should follow the existing visual mockup as the primary visual target.

## Principles
- Stylized, readable, toy-soldier feel
- Mobile landscape first
- High camera with clear battlefield overview
- Chunky fortress silhouettes
- Strong blue vs red readability
- Breach is the main spectacle moment

## Prototype approach
- Use placeholders when needed
- Prioritize readability and gameplay clarity over polish
- If rough layout notes conflict with the mockup, the mockup wins
