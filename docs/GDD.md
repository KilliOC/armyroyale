# Army Royale GDD

The canonical game design document currently lives in Mission Control knowledge-base and should be treated as the primary design source.

Current reference:
- Army Royale — Game Design Document v0.2 (Prototype)

Core idea:
- Landscape real-time multiplayer battlefield
- Two sides pushing a living front line
- Goal is to breach the enemy fortress wall
- Spectacle comes from massed armies, shifting pressure, and a strong breach payoff

This file exists to anchor the repo until the full GDD is copied in or synced here in a later pass.
